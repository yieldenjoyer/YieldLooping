# YieldLooping
Pendle PT/YT Looping Contract with Euler, mints py from USDe -> mints PT/YT on pendle -> sends to euler -> borrows -> loop again an continue
This contract was initially designed to loop Ethereal DEX's eUSDe PT/YT via sending USDe to pendle -> SY USDe -> mint eUSDe -> send PT to euler -> loop again as well as unwind logic. This contract was never fully finished but is close.
